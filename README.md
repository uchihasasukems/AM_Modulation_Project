# AM Modulation Project

## Descripción del proyecto
Este repositorio contiene la implementación de un **sistema de modulación en amplitud (AM)** utilizando Python.  
El objetivo del proyecto es analizar cómo cambia la señal modulada bajo diferentes condiciones de comunicación, incluyendo **ruido, distorsión y atenuación**, evaluando su desempeño en los dominios del tiempo y de la frecuencia.

---

## Contenido del repositorio
- `AM_Modulation.py` – Código principal del proyecto para generar la señal modulada.  
- `Gráficas/` – Carpeta que contiene todas las gráficas generadas durante la simulación.  
- `README.md` – Este archivo explicativo.  
- `Documentación/` – Archivo con el análisis detallado de la actividad.
